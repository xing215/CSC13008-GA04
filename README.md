# Todo App - CSC13008-GA04

A simple client-side rendering (CSR) todo application built with React, TypeScript, and Vite.

## Features

- **Client-side rendering** using React and TypeScript
- **Responsive design** with TailwindCSS
- **LocalStorage** for data persistence
- **Three pages**: Homepage, Todo List, Configuration

## How to Run

1. Install dependencies: `npm install`
2. Start the development server: `npm run dev`
3. Open the provided local URL in your web browser
